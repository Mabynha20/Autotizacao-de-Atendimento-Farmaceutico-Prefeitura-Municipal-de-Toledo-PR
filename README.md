# Autotizacao-de-Atendimento-Farmaceutico-Prefeitura-Municipal-de-Toledo-PR
Sistema em Excel com VBA para apoio à decisão clínica farmacêutica, baseado no protocolo do CFF. Permite cadastrar pacientes, registrar queixas, sugerir tratamentos farmacológicos e não farmacológicos e gerar receituário automaticamente, com armazenamento estruturado dos dados para análises futuras.
